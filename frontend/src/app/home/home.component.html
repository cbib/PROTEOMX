ProteomiX from data to knowledge


Study of protein expression data, in particular relative quantification data in label-free proteomics obtained by high-speed mass spectrometry, has become widely available and democratized within the scientific community, thanks in particular to boom in this discipline in recent years. This type of data today allows access to precise mapping of the biological processes involved in a cell, a cell type, an organism or a population. High-throughput proteomics is now widely used for protein expression profiling, but the bioinformatics analysis of resultant data can be time-consuming and challenging, especially for biologists. We aim to streamline the bioinformatic analyses of protein-level data by developing a user-friendly, interactive web application for exploratory data analysis and visualization, differential expression, enrichment and pathway analysis.


Project description

To meet these needs, we have developed an easy-to-use web application ProteomiX designed to helps biologists easily process and explore Proteomics data, for in-depth analysis of protein expression data. ProteomX encapsulates useful R and python packages, annotation databases, and related web services. The input is a protein-level expression matrix obtained from Proteome Discoverer 2, Progenesis, MaxQuant, or other platforms. Main functionalities include (1) seamless guidance to the user to fill in relevant meta-data, (1) data pre-processing, (2) exploratory data analysis, (3) differential expression, (4) enrichment, (5) pathway analysis and visualization (6) data integration.
